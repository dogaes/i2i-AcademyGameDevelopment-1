# Lane Dodger

A 3D endless-runner obstacle-avoider built with Vanilla Three.js and Vite.

## Controls

- Left/Right or A/D — switch lanes
- Up/Space/W — jump

Red walls must be dodged by changing lanes (too tall to jump over). Orange bars must be jumped.

## Setup

npm install
npm run dev

Then open the printed localhost URL.

## Build

npm run build

## Tech

- Three.js — 3D rendering
- Vite — dev server / bundler
- Vanilla JavaScript, no framework
